Bienvenue au Memory Games

A propos

Le Memory Game est un jeu de mémoire simple et amusant qui met votre concentration ainsi que vos capacités de mémorisation à l’épreuve.

Le principe est facile : retournez deux cartes à la fois et tentez de retrouver les paires identiques.


Titre : Memory Games

Genre : Jeu de mémoire, casse-tête, stratégie
